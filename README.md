1、初始化工程
    // 从git 迁下工程
    git clone https://github.com/my9527/U-I.git
    // 进入工程
    cd U-I
    // 安装npm 插件
    npm install

2、开发命令说明

    2.1、 初始化cordova 工程
            gulp app-init

    2.2、 浏览器调试
            gulp

    2.3、 生成apk
            gulp build

3、提交格式说明
    命令          命令说明
    add             添加新文件
    modified        修改文件
    setup           初始化模块
    delete          删除文件
    fixed           修复bug
